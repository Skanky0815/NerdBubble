    TSW3 wurde offiziell angekündigt. Release: 06.09.2022



    Features im Coreupgrade:


    • Dynamisches Wetter in Services/Journeys sowie entsprechende Änderung der Fahrverhältnisse
    • Bügelfeuer
    • Training Center basierend auf der deutschen Teststrecke Wegberg-Wildenrath als einen zentralen Ort für Tutorials
    • Neue Skybox/ToD System mit volumetrischen Wolken (3D Himmel) und Schattenwurf
    • Freie Wahl des Startpakets
    • Passagiere auf Bahnhöfen passen ihre Kleidung an die Wetterwechsel an
    • Fahrpläne sind Jahreszeitabhängig
    • Neues, sauberes User Interface
    • Lichtverhältnisse nun realistischer sowie "Eye Adaption"
    Preise

    Bestehende TSW2 DLC werden automatisch Freigeschaltet und können in TSW3 benutzt werden.
